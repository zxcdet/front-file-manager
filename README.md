## Installation
Download Node.js 20 or higher version

Run commands:
```bash
$ npm install
```

## Docker
```bash
$ docker-compose up --build
```

## Start server
1. Dev: `npm run start`
2. Prod: `npm run start:prod`
3. Build `npm run build`

Navigate to webpack `http://localhost:8080`

Navigate with start docker-nginx `http://localhost:4200`
